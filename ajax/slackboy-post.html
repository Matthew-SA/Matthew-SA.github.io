Super SlackBoy

Instant messaging application inspired by Slack.

User Authentication

Authentication messures across frontend, backend, and WebSockets.

Instant Messaging

WebSockets update users instantly with new messages across multiple channels.

Edit Messages

Users can update their own messages, which are then transmitted to others in the same channel.

Channel Browser

Browse all publicly available channels, with descriptions and user counts.

Channel Creation

Create a new channel that is immediately available for others to participate in.

Project Details

Super SlackBoy is an instant messaging application inspired by Slack. The application supports instant messaging across multiple secure channels.
